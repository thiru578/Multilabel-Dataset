# Multilabel-Dataset

1) The dataset has 
            - 445 Instances
            - 82 method metrics (description find in http://essere.disco.unimib.it/reverse/MLCSD.html)

2) After pre-processing, we have removed some of the metrics: Now dataset has

           - 445 Instances
            - 46 metrics (20 are class+ other are customized metrics)
